Design Pattern
    ถูกแบ่งออกเป็น 3 กลุ่ม คือ
        1. Creational   เป็นกลุ่มของการออกแบบสำหรับสร้าง object จาก class ใช้เพื่อแก้ปัญหาการสร้าง object 
        2. Structural   เป็นกลุ่มของการออกแบบสำหรับจัดการกับคลาสและวัตถุ เพื่อให้ระบบสามารถขยายและเปลี่ยนแปลงได้ง่ายขึ้น
        3. Behavioral   เป็นกลุ่มของการออกแบบสำหรับจัดการวิธีที่ object จะทำงานร้วมกัน ทำให้สามารถปรับปรุงการทำงานของชุดคำสั่งได้โดยไม่ต้องแก้ไขเยอะ
